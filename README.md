# Cognizant Digital Nurture 5.0 – Java Full Stack Engineer (React)

This repository contains my complete learning journey, hands-on exercises, assignments, notes, and mini-projects completed as part of the **Cognizant Digital Nurture 5.0 (DN 5.0)** training program for the **Java Full Stack Engineer (React)** track.

The repository is organized week-wise and covers core Java, Spring Framework, Spring Boot, Hibernate, REST APIs, Microservices, React, and Data Structures & Algorithms along with all mandatory Cognizant hands-on exercises.
